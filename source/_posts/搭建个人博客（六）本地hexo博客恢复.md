- #### 在本地blog文件夹中,使用Git Bath 运行

  ```java
  npm install hexo
  npm install
  hexo g
  hexo s 
  ```

- #### 继续编写博客，进行hexo d 上传上传失败执行

  ```java
  npm install hexo-deployer-git --save   之后再上传
  hexo d    上传
  ```
