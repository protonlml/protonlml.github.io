###  1.数字，获取其个位 十位 百位 千位

```java
 /**
  * 任意输入一个数字，获取其个位 十位 百位 千位
  */
 Scanner scanner = new Scanner(System.in);
 System.out.println("请输入一个数字");
 int in = scanner.nextInt();
 //获取个位：除1模10
 int ge = in % 10;
 //获取十位：除10模10
 int shi = in / 10 % 10;
 //获取百位：除100模10
 int bai = in/100%10;
 System.out.println("个位"+ge+"十位"+shi+"百位"+bai);
```
